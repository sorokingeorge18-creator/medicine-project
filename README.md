# Мед. документы — Травматология / Ортопедия

Генератор медицинской документации: дневники наблюдения и предоперационный эпикриз по данным пациента, с экспортом в DOCX и ИИ-проверкой грамматики (Claude API).

## Возможности

- Автоматический расчёт дат дневников (день поступления, ПН/СР/ПТ, послеоперационный день, день выписки)
- Генерация дневников наблюдения и предоперационного эпикриза по эталонному формату стационара
- Автоматическое склонение (предложный падеж фиксации, творительный падеж фамилии заведующего)
- Уникальные показатели жизнедеятельности для каждого дневника
- Редактирование документов с проверкой грамматики через Claude API
- Экспорт в DOCX (Times New Roman 12pt, полуторный интервал, поля по ГОСТ)
- Сохранение случаев в localStorage (до 20)

## Запуск

```bash
npm install
npm run dev        # режим разработки (Vite)
```

Продакшен:

```bash
npm run build      # tsc + vite build → dist/
npm start          # Express-сервер: статика из dist/ + /api/grammar
```

Для кнопки «Грамматика» нужна переменная окружения:

```bash
export ANTHROPIC_API_KEY=sk-ant-...
```

## Стек

- React 18 + TypeScript + Vite
- Tailwind CSS
- docx + file-saver (экспорт DOCX)
- date-fns
- Express + @anthropic-ai/sdk (сервер, деплой на Railway)
